# tb4_status_interface
This ROS2 package provides the ROS2 format structure used in [`tb4_status` repository (Custom ROS2 Package)](https://github.com/bmanara/tb4_status) and [`my_tb4_nav` repository (Custom ROS2 Package)](https://github.com/bmanara/my_tb4_nav).

### Installation Guide
`git clone` this repository into your desired ROS2 workspace. Then, `colcon build`.

No setup required (no need to change code), unlike the other packages above.

Refer to the other repositories README for instructions on how to setup, use and their respective developer guides.
